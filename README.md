# csv-sensei
This app enables you to upload csv files and interact with its data in a chat format with `openai/gpt-3.5-turbo`.

### Specs
- client - React, tailwind, Ant Design
- server - Django
- model - openai/gpt-3.5-turbo

## To get started
client:
```
npm run start
```

server:
```
py manage.py runserver
```
